=== AutoClicker / BongoBeat 自动触发工具使用说明 (C# 版本) ===

本仓库包含两个工具：
- AutoClicker：自动模拟鼠标左键点击
- BongoBeat：自动模拟键盘按键（默认 Left Alt，可自定义）

────────────────────────────────────────

【AutoClicker】

编译方法:
  双击运行 build.bat

使用说明:
  1. 双击运行 AutoClicker.exe
  2. 在输入框输入每秒点击次数（1-500）
  3. 按 F8 开始/停止自动点击
  4. 按 ESC 退出程序
  5. 红色方块 = 停止状态，绿色方块 = 运行状态

────────────────────────────────────────

【BongoBeat】

编译方法:
  双击运行 build_bongobeat.bat

使用说明:
  1. 双击运行 BongoBeat.exe
  2. 在上方输入框输入每秒触发次数（1-500）
  3. 在下方输入框输入要模拟的键名（默认 Alt，支持 F1、Space、A 等任意键名，无效输入自动回退为 Alt）
  4. 按 F8 开始/停止自动触发
  5. 按 ESC 退出程序
  6. 红色方块 = 停止状态，绿色方块 = 运行状态

────────────────────────────────────────

【共同优势】

- 纯C#原生代码，性能更好
- 编译后单个EXE文件，无需依赖
- 文件体积小，启动速度快，资源占用低
- F8 / ESC 均为全局热键，无需切换窗口即可操作

【BongoBeat 额外优势】

- 可自定义触发键，灵活适配不同场景
- 默认使用 Left Alt，相比 Ctrl 不会触发滚轮缩放或常用快捷键，日常办公时几乎不影响正常使用

────────────────────────────────────────

【技术特性】

- Windows Forms UI
- 直接调用 Win32 API（user32.dll）
- AutoClicker 使用 mouse_event 模拟鼠标左键
- BongoBeat 使用 keybd_event 模拟键盘按键
- 多线程触发循环
- 全局热键（RegisterHotKey）
- 窗口拖动支持

【注意事项】

- 窗口可拖动，始终置顶
- 仅支持 Windows 系统
- 需要 .NET Framework 4.0 或更高版本（Windows 7+ 自带）
